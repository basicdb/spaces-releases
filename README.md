# Spaces Releases

This repository contains release artifacts and update metadata for the Spaces app.

## Structure

- `updates/` - Update metadata JSON files for auto-updater
- GitHub Releases - DMG files and installers

## For Users

Download the latest version from [Releases](https://github.com/basicdb/spaces-releases/releases).

The app will automatically check for updates when you use the "Check for Updates" feature.

## For Developers

This repository is automatically updated during the release process. See the main repository for release scripts and documentation.
